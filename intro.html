Introdução ao JQUERY

A libraria JQuery permite realizar as seguintes operações

  • Manipulação de HTML/DOM
  • Manipulação de CSS
  • Atribuição de eventos aos HTML
  • Efeitos e animações
  • AJAX
  • Outra utilidades

Instalação da libraria JQuery

A libraria JQuery é um simples ficheiro Javascript que deverá ser referenciada na secção head do documento HTML, através da tag: <script> . Pode ser referenciado localmente ou via web 

	<script src="js/jquery-2.1.0.min.js"></script>
		
	OU
		
	<script src="http://code.jquery.com/jquery-2.1.0.min.js"></script>

O evento document ready

	$(document).ready(function(){
		alert("Olá estou pronto!");
	});
